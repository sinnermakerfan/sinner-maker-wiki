CONTRIBUTE TO WIKI

Because this wiki is hosted entirely on GitHub, you can add your own custom Sinner by adding a file to our repository!

  1. Export your sinner configuration file.
  2. Go to our GitHub Repository sinners/ folder.
  3. Click Add file → Create new file.
  4. Name it your-sinner-name.json and paste your config data inside it!

Make sure your file includes fields for name, class, creator, and description.
